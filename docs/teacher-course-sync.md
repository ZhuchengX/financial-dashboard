# 老师信息与课程包配置双向同步说明

## 功能概述

系统已实现老师信息和课程包配置的双向自动同步，确保数据一致性。

## 同步机制

### 1. 课程包导入 → 更新老师列表

当通过"课程信息录入"导入课程包时：

- **新老师自动添加**：如果Excel中的"默认老师"不在在库老师列表中，系统自动添加该老师
- **课酬以课程包为准**：如果老师已存在，系统会用Excel中的"默认课酬"更新该老师的基准课酬
- **批量处理**：导入完成后会显示新增和更新的老师数量

**实现位置**：`components/finance/course-preset-management.tsx` (第76-156行)

### 2. 编辑老师信息 → 更新课程包配置

当在"老师信息录入 - 在库老师列表"中编辑老师时：

- **修改老师姓名**：系统自动更新所有使用该老师的课程配置中的"默认老师"字段
- **修改基准课酬**：系统自动更新所有该老师的课程的"默认课酬"字段
- **同步反馈**：保存后会显示同步更新的课程数量

**实现位置**：
- 编辑老师信息：`components/finance/teacher-manager.tsx` (第108-142行)
- 快速编辑课酬：`components/finance/teacher-manager.tsx` (第190-214行)

## 使用流程

### 场景1：首次导入课程包

1. 在"课程信息录入"点击"下载模板"
2. 在Excel中填写课程信息和默认老师、默认课酬
3. 点击"导入配置"上传Excel文件
4. 系统自动将新老师添加到在库老师列表，课酬以Excel为准

### 场景2：调整老师定级和课酬

1. 在"老师信息录入 - 在库老师列表"中找到需要修改的老师
2. 点击课酬旁边的编辑图标，或点击"编辑"按钮
3. 修改课酬后保存
4. 系统自动更新该老师所有课程的课酬配置
5. 在"课程信息录入"点击"导出配置"可保存更新后的配置

### 场景3：导出最新课程配置

1. 在任何时候，在"课程信息录入"点击"导出配置"
2. 导出的Excel文件包含最新的老师姓名和课酬信息
3. 该文件可用于备份或再次导入

## 数据流向

```
课程包Excel
    ↓ 导入
在库老师列表 ←→ 双向同步 ←→ 课程配置包
    ↓ 编辑
课程包Excel
    ↓ 导出
```

## 注意事项

- 所有修改都会触发全局数据保存（`markDirty()`）
- 修改会立即生效，影响后续的消课和课酬计算
- 建议定期导出课程配置进行备份
- 老师姓名和课酬的修改会同步到所有关联课程
